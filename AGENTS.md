# Dynamic Instructions for AI Coding Agent

This file contains persistent global rules, project conventions, and instructions that the AI Coding Agent automatically loads.

## Typography & Naming Naming Convention

**Global UI Rule:** 
Throughout the entire website (public site and admin panel), all menu items, page headings, cards, buttons, tabs, forms, and navigation elements must use **Title Case typography**. Never use ALL CAPS labels unless displaying acronyms such as API, SEO, CMS, PDF, or FAQ. Maintain a clean, ministry-focused, professional appearance with consistent typography across every page.

### Examples:

* Dashboard ✅
* Articles ✅
* Users ✅
* Categories ✅
* Settings ✅

### Avoid:

* DASHBOARD ❌
* ARTICLES ❌
* USERS ❌
* dashboard ❌

The UI should follow a modern, professional, and clean content-management-system style similar to contemporary admin panels. Navigation labels should be concise, readable, and human-friendly, using Title Case rather than all-uppercase text.
